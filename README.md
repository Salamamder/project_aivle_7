---

1. 청중의 집중도를 실시간으로 발표자에게 알려준다.
2. 발표가 마무리된 후 발표자에게 청중의 집중도가 시간대별로 어떻게 변화하는지 알려준다.
3. 강의 내용과 청중의 집중도를 기반으로 발표자에게 어떠한 부분이 부족했는지 혹은 좋았는지 피드백을 준다.
